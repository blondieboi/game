# Character Workshop

A tiny static prototype for designing a game character. It uses plain HTML, CSS,
and JavaScript so it can be opened directly in a browser and changed without a
build step.

Open `index.html` to try it. The 3D game view uses Three.js from a CDN, so the
first game launch needs an internet connection.

Current options:

- Character name
- Hair type
- Hair color
- Skin color
- Eye color
- Face expression, Gyokeres-style goal celebration, or mask
- Outfit style
- Outfit badge color
- Background mood
- Random character button
- Start button into a small 3D world
- WASD movement
- Emerald and diamond crystal clusters with simple addition and subtraction challenges
- Swedish challenge text
- Gem counters with icons and respawning crystal clusters
